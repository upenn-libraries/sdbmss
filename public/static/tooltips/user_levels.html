New SDBM User Levels:

  • Contributor can: create Entries, Sources, Names, and Manuscript Records; add comments; and link Entries. Contributors can edit and manage only the Entries, Sources, Names, and Manuscript Records they create.
  • Editor can: do all things that Contributors can do, plus edit all Manuscript Records, manage Names, and approve all Entries, Sources, Names, and Manuscript Records.
  • Super Editor can: do all things that Editors can do; plus, edit legacy data (entries added to the database before October 15, 2015)
  • Admin (Administrators) are: New SDBM staff who can do all things that Contributors and Editors can do, plus edit and delete all Entries, Sources, Names, and Manuscript Records.
