Institution/Collection

Enter name of institution or private collector holding or receiving a manuscript in a given transaction. Use if data comes from an institutional catalogue or if the institution or collector is a recipient in the case of a non-published transaction, or is an institution or collector identified in a secondary source. Enter name as given in data source, even if the name has since changed. For abbeys, monasteries, convents, churches, etc., give city name and institution type or name. For individuals, use standards given in SELLER2.
e.g.:    British Library
  Boston Medical Library
  Manchester, John Rylands Library
  Aulne, Abbey
  Hoe, Robert
  Eton College, Cambridge University

Note: Render all accented letters as the equivalent without diacritical marks. e.g., à / á / â ⇒ a, æ ⇒ ae, ä ⇒ ae, ç ⇒ c, è / é / ê ⇒ e, ì / í / î ⇒ i, œ ⇒ oe, ö ⇒ oe, ü ⇒ ue, etc.
