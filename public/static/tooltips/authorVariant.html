Variant Author

Enter the name exactly as given in the data source. This should be an verbatim quotation from the text, repeating the original idiomatic spelling, name order, and even case usage (usually genitive). If a name is not given in the source, leave Variant Name blank, even if the author can otherwise be inferred; in this case, supply the inferred name in the Author field instead, surrounded by square brackets. The label "Anonymous" should be used only when given exactly as such in the data source.
e.g.:    Gualteri Burlaei
  Anonymous from Verona

Note: Render all accented letters as the equivalent without diacritical marks. e.g., à / á / â ⇒ a, æ ⇒ ae, ä ⇒ ae, ç ⇒ c, è / é / ê ⇒ e, ì / í / î ⇒ i, œ ⇒ oe, ö ⇒ oe, ü ⇒ ue, etc.
