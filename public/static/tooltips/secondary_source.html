Source

Secondary source of transaction data. This field should be used only when identifying source of data imported from an electronic source, such as "Jordanus", or data taken from printed source that records data from a primary transaction source.
e.g.:    Jordanus
  De Ricci, vol. II, p. 2043, no. 104
  Kristeller, vol. IV, p. 121

Senate House De Ricci Index Catalogue Citation. Enter in SOURCE as in the following. (However, if a manuscript record is taken directly from the file, enter the following instead in CAT ID.)
UL, Senate House Library De Ricci Index, Rep., [name of institution as given in index]
UL, Senate House Library De Ricci Index, Coll., [name of collector as given in index]

Note: Render all accented letters as the equivalent without diacritical marks. e.g., à / á / â ⇒ a, æ ⇒ ae, ä ⇒ ae, ç ⇒ c, è / é / ê ⇒ e, ì / í / î ⇒ i, œ ⇒ oe, ö ⇒ oe, ü ⇒ ue, etc.
