Enter search terms into the search box below to bring up possible matches to link. Click Show more search fields beneath the default search box to reveal additional search fields. Click the Show potential links button to automatically search the SDBM for possible links. A pop-up window will appear if no potential links are found. Click “OK” to return to linking tool. When you find matches, click the Add this entry to manuscript option in the far left column of the search result to add the entry to the entry queue above. Entries that are already linked to a Manuscript Record will not display an option to link to an SDBM_MS record. If you think an entry has been incorrectly linked, go to the manuscript record by clicking on the link in the Manuscript column for further research.
