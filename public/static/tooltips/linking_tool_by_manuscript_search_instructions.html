Enter search terms into the search box below to find possible matches to link. Click + add term beneath the default search box to reveal additional search fields. When you find matches, click the Add this entry to manuscript option in the far left column of the search result to add the entry to the entry queue above. Entries that are already linked to a Manuscript Record will not display an option to link to an SDBM_MS record. If you think an entry has been incorrectly linked, go to the manuscript record by clicking on the link in the Manuscript column for further research.
