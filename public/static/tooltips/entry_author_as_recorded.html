Author As Recorded

Enter the name exactly as given in the data source. This should be a verbatim quotation from the text, repeating the original idiomatic spelling, name order, abbreviation, and even case usage (usually genitive). If a name is not given in the source, leave Author As Recorded blank, even if the author can otherwise be inferred: in this case, supply the inferred name in the Author field instead and note it as such with the appropriate certainty symbol in the box next to the Author name. The label “Anonymous” should be used only when given exactly as such in the data source.

e.g.:
Gualteri Burlaei
Anonymous from Verona
