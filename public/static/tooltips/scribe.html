Scribe

Enter name of scribe. Enter name as given in source if only a first name, part of a name, or an abbreviated name. If first and last names are recorded or known, enter last name first. Descriptive notes about the scribe that do not contain an explicit name, such as, "In a contemporary hand" or "Several hands", should be entered in COMMENTS.
e.g.:    Haytzer, Johannes
  Abbot Ellinger
  Fr.Me.Fa.Pe.De.
  Johannes

Note: Render all accented letters as the equivalent without diacritical marks. e.g., à / á / â ⇒ a, æ ⇒ ae, ä ⇒ ae, ç ⇒ c, è / é / ê ⇒ e, ì / í / î ⇒ i, œ ⇒ oe, ö ⇒ oe, ü ⇒ ue, etc.
