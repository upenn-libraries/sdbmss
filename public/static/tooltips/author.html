Author

Enter name as found in Author Lookup Table, which is automatically suggested in the text box. With slight variations, author names follow the Library of Congress Authorities database for authors. Unlike the LOC, do not use commas between names and home towns or between names and common descriptors. Spell out common abbreviations of given names.
e.g.:      Ludolf, von Sachsen    ⇒    Ludolf von Sachsen
    Albertus, Magnus, Saint    ⇒    Albertus Magnus, Saint
    Hyginus, C. Julius    ⇒    Hyginus, Gaius Julius

Note other authorial roles and activities using the following codes in parentheses after the author's name: Attributed = (Attr), Commentator = (Com), Compiler = (Comp), Editor = (Ed), Glossator = (Gl), Introduction = (Intr), Preface = (Pref), Translator = (Tr). Multiple terms should be listed alphabetically.
e.g.:      Jerome, Saint (Ed) (Pref) (Tr)

For pseudonymous authors, use the prefix "Pseudo-" before the authority version of the original author name.
e.g.:      Pseudo-Bonaventure, Saint, Cardinal

Where the data source itself expresses uncertainty, the name should be followed directly, without spaces, with a "?" mark. Where the name is inferred at data entry and not explicitly given in the data source, the name should be enclosed in square brackets, as below.
e.g.:      Bradwardine, Thomas?         doubt expressed in data source
    [Bradwardine, Thomas]   best guess by data entry person

The name "et al" is the official authority designation for multiple unlisted authors.

Note: Render all accented letters as the equivalent without diacritical marks. e.g., à / á / â ⇒ a, æ ⇒ ae, ä ⇒ ae, ç ⇒ c, è / é / ê ⇒ e, ì / í / î ⇒ i, œ ⇒ oe, ö ⇒ oe, ü ⇒ ue, etc.
