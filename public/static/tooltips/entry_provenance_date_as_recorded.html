Provenance Date As Recorded

Enter the date exactly as given in the data source. This should be verbatim from the text.

e.g.:
14th century
June 2005
November 27th, 1958

You have the option of including both the start, end, and any associated dates of possession associated with a provenance agent. Add as many new date fields as needed to capture all data provided in your source. Click on the box to the right of each date field to cycle through the following options:

  • Start = the date a provenance agent began possessing the manuscript
  • End = the date a provenance agent ceased possessing the manuscript
  • Associated = Any date specifically associated with ownership. For example, a date found next to an owner's name written inside the manuscript.
