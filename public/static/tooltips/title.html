Title

Enter actual titles in the language of the original text, e.g. enter "Consolatio philosophiae" NOT "Consolation of Philosophy". If an English or non-English catalogue gives a translation of an original title, for example, enter the standard Latin title given in the Library of Congress Title Authority. If a text is a medieval translation, give the Latin name of the text first, then the translated title in parentheses, e.g. Speculum historiale (Miroir historiale) .

If a non-English catalogue provides only a description of contents or a generic title (a term that indicates a category of books rather than a specific text), give the English equivalent of the description or generic title.
e.g.:    Bible, Book of Hours, Gradual, Missal, etc.

If no title is given but it is obvious what the text would be, enter title in brackets, e.g. if Valerius Maximus is given as an author but no title is given, add the title in brackets: [Facta dictaque memorabilia]. If not entirely sure, add "?" at the end of title.

Note: Render all accented letters as the equivalent without diacritical marks. e.g., à / á / â ⇒ a, æ ⇒ ae, ä ⇒ ae, ç ⇒ c, è / é / ê ⇒ e, ì / í / î ⇒ i, œ ⇒ oe, ö ⇒ oe, ü ⇒ ue, etc.
