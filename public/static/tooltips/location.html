Current Location

Enter current location of manuscript, if known, including shelfmark, if known.

If the location is an institution, enter the city (and US state abbreviation if unclear), accepted short name of institution, and shelf mark.
e.g.:    New York, Morgan Library, M.111
  London, British Library, Add. MS 83
  Paris, BNF, MS. fr. 9220

If a known private collection, enter name only of collector and shelfmark, if supplied.
e.g.:    Lawrence J. Schoenberg Coll., LJS 60

If location is unknown, leave blank.

Note: Render all accented letters as the equivalent without diacritical marks. e.g., à / á / â ⇒ a, æ ⇒ ae, ä ⇒ ae, ç ⇒ c, è / é / ê ⇒ e, ì / í / î ⇒ i, œ ⇒ oe, ö ⇒ oe, ü ⇒ ue, etc.
