Selling Agent

Enter name of auction house, dealer, collector, collection, etc. responsible for Auction/Sale Catalog and Online-only Auction or Bookseller Website. If an individual dealer or seller’s firm is identified by name, give the legal name of the firm. If a private individual seller, give last name first. If a donation, enter the individual’s name. Authority Names already entered into the database will appear in the Selling Agent Name Authority as you type. If a name is new, i.e. not in the Name Authority, create an Authority name by clicking on the Find or Create Authority Name option. This action will open a pop-up window where you can import an Authority name in VIAF. If no VIAF file for a name exists, create a standard manually using the SDBM Personal Name Standard:

Last name, First Name, Title, Life Dates

Do not use commas between names and hometowns or between names and common descriptors. Spell out common abbreviations of given names.

e.g.:
Boston Medical Library
Manchester, John Rylands Library
Aulne, Abbey
Hoe, Robert
Eton College, Cambridge University

Enter any additional details about the person or institution in the Other Info field.
